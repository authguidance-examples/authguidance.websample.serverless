LOCAL EXECUTION
---------------
sls invoke local -f get-transactions -p=..\transactions.json
sls invoke local -f put-profile -p=..\profile.json
sls invoke local -f post-enrol -p=..\enrol.json



SETUP AND COMMANDS
------------------
* sudo npm install serverless -g
* sls offline start then browse to http://localhost:3000
* serverless config credentials --provider aws --key mykey --secret mysecret --overwrite
* sls deploy -v
* sls invoke -f any -l to run the lambda

Deploy the API Gateway - URL is then of this form:
https://{restapi-id}.execute-api.{region}.amazonaws.com/{stageName}
https://yky9lwpz8j.execute-api.us-east-1.amazonaws.com/dev/companies
